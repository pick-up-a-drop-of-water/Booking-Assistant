# 场地自动预定助手

华科西体羽毛球场地自动预定工具，使用 python tkinter 实现图形界面。

支持 Windows、Linux，python 依赖见 `requirement.txt`。

在[这里](https://github.com/rachpt/Booking-Assistant/releases)直接下载二进制单文件。
